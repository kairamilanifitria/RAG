Data Cleaning sebelum visualisasi di Tableau
Kenapa perlu data cleaning di Python sebelum visualisasi di Tableau?
Tableau kuat di visualisasi, tapi terbatas di data prep kompleks.
Data mentah biasanya kotor: ada missing values, duplikasi, outlier, format tanggal yang salah, atau kategori yang tidak konsisten.
Python (pandas, numpy) jauh lebih fleksibel untuk membersihkan dan menyiapkan data sebelum divisualisasikan di Tableau.
